# 📇 Apex CRM Pro – Contact Management System

## 📌 Project Overview
Apex CRM Pro is a professional Contact Management System developed in Python.  
It allows users to manage contacts efficiently using both:

- 🖥 Streamlit GUI (Graphical Interface)
- 💻 CLI (Command Line Interface)

This system simulates a mini Customer Relationship Management (CRM) tool with advanced features like search, filters, update, and data persistence.

---

## 🚀 Features

### 🔹 Core Features
- Add new contacts with validation
- View all contacts in table format
- Search contacts by:
  - Name (partial match)
  - Phone
  - Email
- Filter contacts by:
  - City
  - Company
- Update existing contact details
- Delete contacts by:
  - ID
  - Name

---

### 🔹 Advanced Features
- Unique ID auto-generation
- Case-insensitive search
- Data stored in JSON file
- Export contacts to CSV
- Import contacts from CSV
- Sorting options (Name, Company, Latest)
- Exception handling

---

### 🌟 Extra Features
- Dual Interface:
  - Streamlit UI (User-friendly)
  - CLI Mode (Terminal-based)
- Clean and modular code structure
